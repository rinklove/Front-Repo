CSS 선택자

        선택자란?
        특정 HTML 요소를 선택할 때 사용

1. 모든 요소를 선택하는 선택자: *

        * {
            속성: 값;
            속성: 값;
            속성: 값;
        }

2. 태그 선택자

        HTML 문서 내 같은 태그를 모두 선택
        태그 {
            속성: 값;
            속성: 값;
            속성: 값;
        }

3. 클래스 선택자: .(클래스명)

        HTML문서 내 여러개의 태그를 동일한 클래스명으로 지정하여 선택
        .(클래스명) {
            속성: 값;
            속성: 값;
            속성: 값;
        }

4. 아이디 선택자: #(아이디명)

        HTML문서 내 하나의 태그를 아이디명으로 지정하여 선택
        #(아이디명) {
            속성: 값;
            속성: 값;
            속성: 값;
        }
